## HR01 身份認證管理 (IAM) - 詳細進度

### 已完成項目

#### 1. API 層
- [x] `AuthApi.ts` - 認證 API (登入、登出、SSO、密碼變更)
- [x] `UserApi.ts` - 使用者管理 API (CRUD、批次操作、密碼重置)
- [x] `RoleApi.ts` - 角色管理 API (CRUD、權限管理)
- [x] `AuthTypes.ts` - 完整的 DTO 類型定義

#### 2. 元件層
- [x] `LoginForm.tsx` - 登入表單 (含 SSO 按鈕)
- [x] `UserTable.tsx` - 使用者列表表格
- [x] `UserFormModal.tsx` - 使用者新增/編輯對話框
- [x] `ResetPasswordModal.tsx` - 管理員重置密碼對話框
- [x] `RoleCard.tsx` - 角色卡片元件
- [x] `RoleFormModal.tsx` - 角色新增/編輯對話框
- [x] `PermissionTree.tsx` - 權限樹選擇元件

#### 3. Hooks 層
- [x] `useLogin.ts` - 登入邏輯
- [x] `useUsers.ts` - 使用者管理邏輯
- [x] `useRoles.ts` - 角色管理邏輯

#### 4. Factory 層
- [x] `UserViewModelFactory.ts` - DTO 轉換工廠
  - `createProfileFromDTO()` - 登入使用者資料轉換
  - `createUserViewModelFromDTO()` - 使用者管理資料轉換
  - `createUserListItem()` - 使用者列表項目轉換
  - `createRoleViewModelFromDTO()` - 角色資料轉換
  - `createPermissionTreeData()` - 權限樹資料轉換

#### 5. 頁面層
- [x] `HR01LoginPage.tsx` - 登入頁面
- [x] `HR01UserManagementPage.tsx` - 使用者管理頁面
- [x] `HR01RoleManagementPage.tsx` - 角色權限管理頁面
