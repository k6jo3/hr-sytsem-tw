## 1. 基類繼承關係

```
BaseTest (通用配置)
    │
    ├── BaseUnitTest (純單元測試)
    │       │
    │       ├── BaseDomainTest (Domain Entity 測試)

            ├── BaseApiContractTest (API 契約)
            └── BaseContractTest (合約驅動測試)
```

## 2. BaseServiceTest

Service 層測試基類，提供 QueryGroup 攔截與快照比對能力。

**位置:** `hrms-common/src/test/java/.../test/base/BaseServiceTest.java`

```java
public abstract class BaseServiceTest<S> {

    @Mock
    protected BaseRepository<?> repository;

    protected ObjectMapper snapshotMapper;

    /**
     * 驗證 Service 產出的 QueryGroup
     */
    protected void verifyQuery(Runnable serviceCall, String snapshotName) {
        ArgumentCaptor<QueryGroup> captor = ArgumentCaptor.forClass(QueryGroup.class);
        
        serviceCall.run();
        
        verify(repository, atLeastOnce()).findPage(captor.capture(), any());
        
        SnapshotUtils.compareOrUpdate(snapshotName, captor.getValue());
    }

    /**
     * 驗證計算結果
     */
    protected <T> void verifyResult(T result, String snapshotName) {
        SnapshotUtils.compareOrUpdate(snapshotName, result);
    }
}
```

## 3. BaseContractTest

合約驅動測試基類，驗證 API 是否符合 SA 規格。

**位置:** `hrms-common/src/test/java/.../test/base/BaseContractTest.java`

```java
@SpringBootTest(webEnvironment = WebEnvironment.MOCK)
@AutoConfigureMockMvc
public abstract class BaseContractTest {

    @Autowired
    protected MockMvc mockMvc;

    @Autowired
    protected ObjectMapper objectMapper;

    @Autowired
    protected MarkdownContractEngine contractEngine;

    /**
     * 載入合約規格
     */
    protected String loadContractSpec(String serviceName) {
        return FileUtils.readString(
            "knowledge/05_Testing_Standards/Contracts/" + serviceName + "_contracts.md"
        );
    }

    /**
     * 驗證 API 是否符合合約
     */
    protected void verifyApiContract(String url, Object request, 
                                     String contract, String scenarioId) throws Exception {
        // 執行 API 並攔截 QueryGroup
        QueryGroup actualQuery = executeAndCapture(url, request);
        
        // 驗證合約
        contractEngine.assertContract(actualQuery, contract, scenarioId);
    }
}
```

## 4. 使用範例

### Service 測試

```java
class GetEmployeeListServiceTest extends BaseServiceTest<GetEmployeeListServiceImpl> {

    @InjectMocks
    private GetEmployeeListServiceImpl service;

    @Test
    void searchByDepartment_ShouldMatchSnapshot() {
        GetEmployeeListRequest req = new GetEmployeeListRequest();
        req.setDepartmentName("研發部");

        verifyQuery(
            () -> service.getResponse(req, mockUser),
            "employee_search_by_dept"
        );
    }
}
```

### 合約測試

```java
class EmployeeContractTest extends BaseContractTest {

    @Test
    void searchEmployee_ShouldMatchContract() throws Exception {
        String contract = loadContractSpec("employee");

        verifyApiContract(
            "/api/v1/employees/search",
            new EmployeeSearchRequest(),
            contract,
            "EMP-001"
        );
    }
}
```
