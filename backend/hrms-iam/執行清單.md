## 已完成 API 端點

### 認證 API (/api/v1/auth)

| 方法 | 路徑 | 說明 | Service |
|:---:|:---|:---|:---|
| POST | /login | 使用者登入 | LoginServiceImpl |
| POST | /refresh | 刷新 Token | RefreshTokenServiceImpl |
| POST | /logout | 使用者登出 | LogoutServiceImpl |
| POST | /password/reset | 使用者變更密碼 | ResetPasswordServiceImpl |
| POST | /users/{userId}/password/reset | 管理員重設密碼 | AdminResetPasswordServiceImpl |

### 角色管理 API (/api/v1/roles)

| 方法 | 路徑 | 說明 | Service |
|:---:|:---|:---|:---|
| GET | / | 查詢角色列表 | GetRoleListServiceImpl |
| GET | /{roleId} | 查詢單一角色 | GetRoleServiceImpl |
| GET | /system | 查詢系統角色 | GetSystemRolesServiceImpl |
| POST | / | 建立角色 | CreateRoleServiceImpl |
| PUT | /{roleId} | 更新角色 | UpdateRoleServiceImpl |
| PUT | /{roleId}/activate | 啟用角色 | ActivateRoleServiceImpl |
| PUT | /{roleId}/deactivate | 停用角色 | DeactivateRoleServiceImpl |
| DELETE | /{roleId} | 刪除角色 | DeleteRoleServiceImpl |
| POST | /{roleId}/permissions | 指派權限 | AssignPermissionsServiceImpl |
