## 績效考核服務 (Performance Service)

### 核心職責
- 考核週期管理（試用期/季度/年度）
- 考核表單彈性設計（KPI、360度評估）
- 考核流程（自評→主管評→覆核→HR確認）
- 績效評等與強制分配曲線
- 績效與薪資連動

### 聚合根
```
PerformanceCycle {
  cycleId: UUID
  cycleName: String (2025年度考核)
  cycleType: CycleType (PROBATION, QUARTERLY, ANNUAL)
  startDate: Date
  endDate: Date
  status: CycleStatus
}

PerformanceReview {
  reviewId: UUID
  cycleId: UUID
  employeeId: UUID
  revieweeId: UUID (被考核者)
  reviewType: ReviewType (SELF, MANAGER, PEER)
  
  evaluationItems: List<EvaluationItem>
  overallRating: String (A/B/C/D or 1-5分)
  comments: Text
  
  status: ReviewStatus
}

EvaluationItem {
  itemName: String (工作品質、團隊合作...)
  weight: Decimal (權重)
  score: Integer (分數)
}
```

### 核心API
```
POST /api/v1/performance/cycles/{cycleId}/reviews
PUT /api/v1/performance/reviews/{id}/submit
GET /api/v1/performance/reports/rating-distribution
```

### 整合
- 訂閱 `EmployeePromoted` (Organization) → 記錄考核結果
- 發布 `PerformanceReviewCompleted` → Payroll (調薪、獎金)

## 招募管理服務 (Recruitment Service)

### 核心職責
- 職缺需求管理與審核
- 應徵者管理（履歷、來源、狀態追蹤）
- 面試管理（排程、評估、結果彙整）
- Offer管理（發放、接受/拒絕）
- 招募成效分析

### 聚合根
```
JobOpening {
  openingId: UUID
  jobTitle: String
  departmentId: UUID
  numberOfPositions: Integer
  salaryRange: String
  requirements: Text
  status: OpeningStatus (OPEN, CLOSED, FILLED)
}

Candidate {
  candidateId: UUID
  fullName: String
  email: String
  phoneNumber: String
  resumeUrl: String
  source: RecruitmentSource (JOB_BANK, REFERRAL, WEBSITE)
  applicationDate: Date
  status: CandidateStatus (NEW, SCREENING, INTERVIEWING, OFFERED, HIRED, REJECTED)
}

Interview {
  interviewId: UUID
  candidateId: UUID
  interviewDate: DateTime
  interviewers: List<UUID>
  evaluations: List<InterviewEvaluation>
  result: InterviewResult (PASS, FAIL, PENDING)
}

Offer {
  offerId: UUID
  candidateId: UUID
  offeredSalary: Decimal
  offerDate: Date
  expiryDate: Date
  status: OfferStatus (PENDING, ACCEPTED, REJECTED, EXPIRED)
}
```

### 核心API
```
POST /api/v1/recruitment/job-openings
POST /api/v1/recruitment/candidates
POST /api/v1/recruitment/interviews/schedule
POST /api/v1/recruitment/offers
```

### 整合
- 發布 `CandidateHired` → Organization Service (建立員工)
- 整合人力銀行API (104, 1111等)

## 訓練管理服務 (Training Service)

### 核心職責
- 訓練計畫管理
- 課程管理（內訓/外訓、線上/線下）
- 課程報名與審核
- 訓練時數追蹤（法定時數合規）
- 證照管理與到期提醒

### 聚合根
```
TrainingPlan {
  planId: UUID
  year: Integer
  budget: Decimal
  courses: List<UUID>
}

TrainingCourse {
  courseId: UUID
  courseName: String
  courseType: CourseType (INTERNAL, EXTERNAL)
  deliveryMode: DeliveryMode (ONLINE, OFFLINE)
  instructor: String
  duration: Decimal (時數)
  maxParticipants: Integer
  startDate: Date
  endDate: Date
  status: CourseStatus
}

TrainingEnrollment {
  enrollmentId: UUID
  courseId: UUID
  employeeId: UUID
  status: EnrollmentStatus (REGISTERED, ATTENDED, COMPLETED, CANCELLED)
  attendance: Boolean
  completedHours: Decimal
}

Certificate {
  certificateId: UUID
  employeeId: UUID
  certificateName: String
  issueDate: Date
  expiryDate: Date (nullable)
  attachmentUrl: String
}
```

### 核心API
```
POST /api/v1/training/courses
POST /api/v1/training/enrollments
POST /api/v1/training/certificates
GET /api/v1/training/certificates/expiring?days=30
```

### 整合
- 發布 `CertificateExpiring` → Notification
- 與Document Service整合（教材管理）
