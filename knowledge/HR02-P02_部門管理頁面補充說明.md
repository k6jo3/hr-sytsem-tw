### 開發步驟
1. **建立頁面組件:** `src/pages/HR02DepartmentListPage.tsx`
2. **建立 Feature 模組:**
   ```
   src/features/organization/
   ├── api/
   │   └── DepartmentApi.ts          # API 調用
   ├── factory/
   │   └── DepartmentViewModelFactory.ts  # DTO 轉換
   ├── components/
   │   ├── DepartmentListTable.tsx   # 表格組件
   │   ├── DepartmentFilterBar.tsx   # 篩選列
   │   └── CreateDepartmentModal.tsx # 新增對話框
   ├── hooks/
   │   └── useDepartmentList.ts      # 列表查詢 Hook
   └── model/

   ```
