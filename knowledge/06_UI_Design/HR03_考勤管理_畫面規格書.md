### 3.3 操作事件規格

| 事件 ID | 觸發元素 | 行為 | 對應 API | UI 反應 |
|:---|:---|:---|:---|:---|
| E-ATT-01 | 頁面載入 | 取得今日班別與打卡記錄 | `GET /api/v1/attendance/today` | 顯示班別與已打卡記錄 |
| E-ATT-02 | 上班打卡按鈕 | 取得 GPS → 打卡 | `POST /api/v1/attendance/check-in` | 成功：顯示打卡時間；遲到：顯示警告 |
| E-ATT-03 | 下班打卡按鈕 | 取得 GPS → 打卡 | `POST /api/v1/attendance/check-out` | 成功：顯示打卡時間 |

### 3.4 狀態與互動

| 狀態 | 打卡按鈕 | 說明 |
|:---|:---|:---|
| 未上班打卡 | 顯示「上班打卡」(primary) | 正常可操作 |
| 已上班打卡 | 顯示「下班打卡」(primary) | 切換為下班 |
| 已下班打卡 | 按鈕禁用 | 今日已完成 |
| 遲到打卡 | 打卡後顯示 `Tag color="warning"` 遲到 | 系統自動判斷 |

## 4. HR03-P02 請假加班申請

### 4.1 頁面概述

| 項目 | 說明 |
|:---|:---|
| **用途** | 員工提交請假或加班申請，查看申請歷史與審核狀態 |

| **進入方式** | 側邊選單 → 考勤管理 → 請假加班申請 |
| **頁面類型** | 列表頁 + 申請 Modal |

### 4.2 Wireframe

```
┌──────────────────────────────────────────────────────┐
│ 請假加班申請                    [請假申請] [加班申請]  │
├──────────────────────────────────────────────────────┤
│ ┌──────────┐ ┌──────────┐ ┌──────────┐              │
│ │ 申請類型 │ │ 狀態 [▼] │ │ 日期範圍 │              │
│ │ [全部 ▼] │ │ [全部  ] │ │ [__~__]  │              │
│ └──────────┘ └──────────┘ └──────────┘              │
│                              [搜尋] [重置]           │
├──────────────────────────────────────────────────────┤
│ ┌────┬──────┬──────┬──────┬──────┬──────┬──────┬───┐│
│ │ #  │ 類型 │ 假別 │ 起迄 │ 時數 │ 狀態 │ 審核 │操作││
│ ├────┼──────┼──────┼──────┼──────┼──────┼──────┼───┤│
│ │ 1  │ 請假 │特休  │03/01 │ 8h  │待審核│王主管│取消││
│ │ 2  │ 加班 │—     │02/28 │ 3h  │已核准│李經理│ — ││
│ │ 3  │ 請假 │病假  │02/20 │ 8h  │已駁回│王主管│重新││
│ └────┴──────┴──────┴──────┴──────┴──────┴──────┴───┘│
├──────────────────────────────────────────────────────┤
│                              第 1-20 筆，共 15 筆    │
└──────────────────────────────────────────────────────┘
```

### 4.3 搜尋/篩選條件

| 條件 | 欄位類型 | 選項來源 | 預設值 | API 參數 |
|:---|:---|:---|:---|:---|
| 申請類型 | `Select` | 請假/加班 | 全部 | `type` |
| 狀態 | `Select` | PENDING/APPROVED/REJECTED/CANCELLED | 全部 | `status` |
| 日期範圍 | `RangePicker` | — | 近 3 個月 | `startDate/endDate` |

### 4.4 操作事件規格

| 事件 ID | 觸發元素 | 行為 | 對應 API | UI 反應 |
|:---|:---|:---|:---|:---|
| E-LV-01 | 請假申請按鈕 | 開啟請假申請 Modal | — | 彈出請假表單 Modal |
| E-LV-02 | 加班申請按鈕 | 開啟加班申請 Modal | — | 彈出加班表單 Modal |
| E-LV-03 | 取消按鈕 | 取消待審核申請 | `PUT /api/v1/leave/applications/{id}/cancel` | Modal.confirm |
| E-LV-04 | 重新按鈕 | 複製已駁回申請 | — | 彈出 Modal，帶入原申請資料 |
| E-LV-05 | Modal 假別選擇 | 即時顯示剩餘額度 | `GET /api/v1/leave/balances/{type}` | 顯示剩餘天數 |
| E-LV-06 | Modal 送出 | 提交申請 | `POST /api/v1/leave/applications` | 成功 → 重新載入列表 |

## 7. HR03-P05 ~ P10 簡要規格

### HR03-P05 加班申請管理

| 項目 | 說明 |
|:---|:---|
| 頁面類型 | 列表頁 |
