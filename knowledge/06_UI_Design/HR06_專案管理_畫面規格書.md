## 7. HR06-P05 專案任務 (WBS)

### 7.1 Wireframe

```
┌──────────────────────────────────────────────────────┐
│ ← 返回     HR系統 - 工項管理          [+ 新增工項]    │
├──────────────────────────────────────────────────────┤
│ ┌────┬─────────────┬──────┬──────┬──────┬──────┬───┐│
│ │層級│ 工項名稱     │ 負責人│預估  │已用  │進度  │操作││
│ ├────┼─────────────┼──────┼──────┼──────┼──────┼───┤│
│ │ 1  │▼ 需求分析    │王小明│ 200h │ 180h │ 90%  │編輯││
│ │ 1.1│  ├ 需求訪談  │李小花│  80h │  80h │100%  │編輯││
│ │ 1.2│  └ 規格撰寫  │王小明│ 120h │ 100h │ 83%  │編輯││
│ │ 2  │▶ 系統設計    │張三  │ 300h │ 150h │ 50%  │編輯││
│ │ 3  │▶ 開發實作    │—     │ 800h │ 200h │ 25%  │編輯││
│ └────┴─────────────┴──────┴──────┴──────┴──────┴───┘│
└──────────────────────────────────────────────────────┘
```

### 7.2 操作事件規格

| 事件 ID | 觸發元素 | 行為 | 對應 API |
|:---|:---|:---|:---|
| E-WBS-01 | 新增工項按鈕 | 開啟 HR06-M03 | — |
| E-WBS-02 | 展開/收合 | 展開子工項 | 前端狀態 |
| E-WBS-03 | 編輯按鈕 | 開啟 HR06-M03 (edit) | `GET /api/v1/projects/{id}/tasks/{taskId}` |
| E-WBS-04 | Modal 確認 | 建立/更新工項 | `POST/PUT /api/v1/projects/{id}/tasks` |
| E-WBS-05 | 拖曳排序 | 調整工項順序 | `PUT /api/v1/projects/{id}/tasks/{taskId}/reorder` |

**WBS 限制規則：** 最多 5 層、父工項層級 +1、允許拖曳調整順序
