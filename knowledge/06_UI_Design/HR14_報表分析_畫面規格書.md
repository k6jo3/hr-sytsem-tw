### 3.3 KPI 指標定義

| 指標 | 計算公式 | 資料來源 |
|:---|:---|:---|
| 在職人數 | 當前 ACTIVE 員工數 | employee_report_view |
| 離職率 | 離職人數 / 期初人數 × 100% | monthly_hr_stats |
| 稼動率 | 計費工時 / 總可用工時 × 100% | project_cost_snapshots |
| 人均成本 | 總人力成本 / 員工人數 | monthly_hr_stats |

### 3.4 操作事件規格

| 事件 ID | 觸發元素 | 行為 | 對應 API |
|:---|:---|:---|:---|
| E-RPT-01 | 日期選擇 | 切換報表期間 | `GET /api/v1/reports/dashboard?month={month}` |
| E-RPT-02 | 匯出按鈕 | 匯出 Excel/PDF | `GET /api/v1/reports/dashboard/export` |
| E-RPT-03 | 圖表點擊 | 鑽取明細 | 導向對應報表頁 |

## 4. HR14-P02 HR 報表

### 4.1 報表項目

| 報表 | 說明 | 圖表類型 |
|:---|:---|:---|
| 員工花名冊 | 全部在職員工列表 | Table + Excel 匯出 |

| 離職分析 | 離職率趨勢、離職原因分布 | 折線圖 + 圓餅圖 |
| 年資分布 | 員工年資區間分布 | 柱狀圖 |
| 學歷統計 | 學歷分布 | 圓餅圖 |

## 5. HR14-P03 專案報表

### 5.1 報表項目

| 報表 | 說明 | 圖表類型 |
|:---|:---|:---|
| 專案成本分析 | 預算 vs 實際成本對比 | 柱狀圖（對比） |
| 稼動率報表 | 各部門/員工稼動率 | 柱狀圖 + 熱力圖 |
| 專案獲利分析 | (合約金額 - 實際成本) / 合約金額 | 表格 + 排名 |
| 工時分配報表 | 各專案工時佔比 | Treemap 圖 |

### 5.2 關鍵計算

| 指標 | 計算公式 |
|:---|:---|
| 專案獲利率 | (合約金額 - 實際成本) / 合約金額 × 100% |
| 稼動率 | 計費工時 / 總可用工時 × 100% |
| 成本效率 | 實際成本 / 預算 × 100% |

## 7. CQRS ReadModel 事件訂閱

| 訂閱事件 | 更新 ReadModel | 更新頻率 |
|:---|:---|:---|
| `EmployeeCreatedEvent` | employee_report_view | 即時 |
| `EmployeeTerminatedEvent` | employee_report_view + 離職率 | 即時 |
| `TimesheetApprovedEvent` | project_cost_snapshots | 即時 |
| `PayrollRunCompletedEvent` | monthly_hr_stats | 即時 |
| RefreshEmployeeViewJob | employee_report_view | 每小時 |
| CalculateProjectCostJob | project_cost_snapshots | 每日凌晨 |
| CalculateMonthlyStatsJob | monthly_hr_stats | 每月 1 日 |
