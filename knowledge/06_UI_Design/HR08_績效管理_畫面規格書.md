## 3. HR08-P01 考核週期管理

### 3.1 Wireframe

```
┌──────────────────────────────────────────────────────┐
│ 首頁 / 績效考核 / 考核週期管理           ← Breadcrumb │
│ 考核週期管理                       [+ 建立考核週期]   │
├──────────────────────────────────────────────────────┤
│ ┌────┬──────────┬──────┬──────┬──────┬──────┬────┐  │
│ │ #  │ 週期名稱 │ 期間 │ 類型 │ 人數 │ 狀態 │操作│  │
│ ├────┼──────────┼──────┼──────┼──────┼──────┼────┤  │
│ │ 1  │2026上半年│01-06 │年度  │ 120  │進行中│設計│  │
│ │    │          │      │      │      │      │啟動│  │
│ │ 2  │2025下半年│07-12 │年度  │ 115  │已完成│查看│  │
│ └────┴──────────┴──────┴──────┴──────┴──────┴────┘  │
└──────────────────────────────────────────────────────┘
```

### 3.2 操作事件規格

| 事件 ID | 觸發元素 | 行為 | 對應 API |
|:---|:---|:---|:---|
| E-PFM-01 | 建立週期按鈕 | 建立考核週期 Modal | `POST /api/v1/performance/cycles` |
| E-PFM-02 | 設計按鈕 | 進入表單設計頁 | — → P02 |
| E-PFM-03 | 啟動按鈕 | 啟動考核週期 | `PUT /api/v1/performance/cycles/{id}/activate` |
| E-PFM-04 | 啟動確認 | 通知所有員工開始自評 | 發布 CycleActivatedEvent |

## 7. HR08-P05 績效報告

| 項目 | 說明 |
|:---|:---|
| 頁面類型 | 儀表板 |
| 主要功能 | 評等分布、部門平均分數、年度趨勢 |
| 圖表 | ECharts 圓餅圖（評等分布）、柱狀圖（部門比較）、折線圖（年度趨勢） |
| 關鍵 API | `GET /api/v1/performance/reports` |
| PageHeader | Breadcrumb: 首頁 / 績效考核 / 績效報告 |
| 空狀態 | 無績效資料時顯示 `Empty.PRESENTED_IMAGE_SIMPLE` + 「尚無績效報告資料」 |
| Card 圓角 | 統計卡片與圖表卡片使用 `borderRadiusLG: 12` |

## 9. API 端點對照表

| 方法 | 端點 | 用途 | 對應畫面 |
|:---|:---|:---|:---|
| `GET` | `/api/v1/performance/cycles` | 考核週期列表 | P01 |
| `POST` | `/api/v1/performance/cycles` | 建立週期 | P01 |
| `PUT` | `/api/v1/performance/cycles/{id}/activate` | 啟動週期 | P01 |
| `GET` | `/api/v1/performance/cycles/{id}/template` | 取得考核模板 | P02 |
| `PUT` | `/api/v1/performance/cycles/{id}/template` | 更新考核模板 | P02 |
| `GET` | `/api/v1/performance/my` | 我的考核 | P03 |
| `PUT` | `/api/v1/performance/reviews/{id}/draft` | 暫存自評 | P03 |
| `PUT` | `/api/v1/performance/reviews/{id}/submit-self` | 送出自評 | P03 |
| `GET` | `/api/v1/performance/team` | 團隊績效列表 | P04 |
| `GET` | `/api/v1/performance/reviews/{id}` | 取得評核詳情 | P04 |
| `PUT` | `/api/v1/performance/reviews/{id}/submit-manager` | 主管評核 | P04 |
| `GET` | `/api/v1/performance/reports` | 績效報告 | P05 |
