## 3. HR12-P01 通知範本管理

### 3.1 Wireframe

```
┌──────────────────────────────────────────────────────┐
│ 通知範本管理                          [+ 新增範本]    │
├──────────────────────────────────────────────────────┤
│ ┌────┬──────────────┬──────┬──────────┬──────┬────┐ │
│ │ #  │ 範本名稱     │ 事件 │ 渠道     │ 狀態 │操作│ │
│ ├────┼──────────────┼──────┼──────────┼──────┼────┤ │
│ │ 1  │請假核准通知  │LEAVE │APP+EMAIL │啟用  │編輯│ │
│ │ 2  │工時審核通知  │TSH   │APP+EMAIL │啟用  │編輯│ │
│ │ 3  │薪資單通知    │PAY   │APP       │啟用  │編輯│ │
│ │ 4  │證照到期提醒  │CERT  │APP+EMAIL │啟用  │編輯│ │
│ └────┴──────────────┴──────┴──────────┴──────┴────┘ │
└──────────────────────────────────────────────────────┘
```

### 3.2 操作事件規格

| 事件 ID | 觸發元素 | 行為 | 對應 API |
|:---|:---|:---|:---|
| E-TPL-01 | 新增範本按鈕 | 建立通知範本 | `POST /api/v1/notification/templates` |
| E-TPL-02 | 編輯按鈕 | 編輯範本內容與渠道 | `PUT /api/v1/notification/templates/{id}` |

### 4.2 操作事件規格

| 事件 ID | 觸發元素 | 行為 | 對應 API |
|:---|:---|:---|:---|
| E-NTF-01 | 頁面載入 | 載入通知列表 | `GET /api/v1/notifications/my` |
| E-NTF-02 | 通知項點擊 | 標記已讀 + 跳轉 | `PUT /api/v1/notifications/{id}/read` |
| E-NTF-03 | 全部已讀 | 批次標記已讀 | `PUT /api/v1/notifications/read-all` |
| E-NTF-04 | 載入更多 | 載入下一頁 | `GET /api/v1/notifications/my?page={n}` |

## 8. API 端點對照表

| 方法 | 端點 | 用途 | 對應畫面 |
|:---|:---|:---|:---|
| `GET` | `/api/v1/notification/templates` | 範本列表 | P01 |
| `POST` | `/api/v1/notification/templates` | 新增範本 | P01 |
| `PUT` | `/api/v1/notification/templates/{id}` | 更新範本 | P01 |
| `GET` | `/api/v1/notifications/my` | 我的通知 | P02 |
| `PUT` | `/api/v1/notifications/{id}/read` | 標記已讀 | P02 |
| `PUT` | `/api/v1/notifications/read-all` | 全部已讀 | P02 |
| `GET` | `/api/v1/notifications/unread-count` | 未讀數 | Header Badge |
| `GET` | `/api/v1/notification/preferences` | 偏好設定 | P03 |
| `PUT` | `/api/v1/notification/preferences` | 更新偏好 | P03 |
