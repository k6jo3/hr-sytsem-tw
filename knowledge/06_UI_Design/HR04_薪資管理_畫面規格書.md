## 3. HR04-P01 計薪作業中心

### 3.1 頁面概述

| 項目 | 說明 |
|:---|:---|
| **用途** | 管理薪資計算批次的完整生命週期 |
| **使用角色** | HR |
| **進入方式** | 側邊選單 → 薪資核算 → 計薪作業中心 |
| **頁面類型** | 列表頁 |

### 3.2 Wireframe

```
┌──────────────────────────────────────────────────────┐
│ 首頁 / 薪資核算 / 計薪作業中心          ← PageHeader  │
│ 計薪作業中心                         [+ 建立薪資批次] │
├──────────────────────────────────────────────────────┤
│ ┌──────────┐ ┌──────────┐ ┌──────────┐              │
│ │ 薪資月份 │ │ 狀態 [▼] │ │ 薪資制度 │              │
│ │ [2026-03]│ │ [全部  ] │ │ [全部 ▼] │              │
│ └──────────┘ └──────────┘ └──────────┘              │
│                              [搜尋] [重置]           │
├──────────────────────────────────────────────────────┤
│ ┌────┬────────┬──────┬──────┬──────┬──────┬────┐    │
│ │ #  │ 批次號 │ 月份 │ 人數 │ 總額 │ 狀態 │操作│    │
│ ├────┼────────┼──────┼──────┼──────┼──────┼────┤    │
│ │ 1  │PR-2603 │03月  │ 120  │$5.2M │草稿  │查看│    │
│ │    │        │      │      │      │      │執行│    │
│ │ 2  │PR-2602 │02月  │ 118  │$5.1M │已發放│查看│    │
│ └────┴────────┴──────┴──────┴──────┴──────┴────┘    │
├──────────────────────────────────────────────────────┤
│                              第 1-20 筆，共 12 筆    │
└──────────────────────────────────────────────────────┘
```

### 3.3 資料欄位規格

| 欄位 | API 來源 | 顯示格式 | 寬度 | 排序 |
|:---|:---|:---|:---|:---|
| 批次編號 | `batchNumber` | 原始值 | 120px | ✅ |
| 薪資月份 | `payrollMonth` | `YYYY年MM月` | 120px | ✅ |
| 計薪人數 | `employeeCount` | 整數 | 80px | — |
| 薪資總額 | `totalAmount` | `$#,###` 千分位 | 120px | ✅ |
| 狀態 | `status` | 批次狀態 Tag | 120px | ✅ |
| 操作 | — | 按鈕群組 | 150px | — |

### 3.4 薪資批次狀態流

```
DRAFT → CALCULATING → COMPLETED → SUBMITTED → APPROVED → RELEASED
 草稿     計算中       計算完成     已送審      已核准      已發放
```

### 3.5 操作事件規格

| 事件 ID | 觸發元素 | 行為 | 對應 API | 可用狀態 |
|:---|:---|:---|:---|:---|
| E-RUN-01 | 建立批次按鈕 | 建立新薪資批次 | `POST /api/v1/payroll-runs` | — |
| E-RUN-02 | 查看按鈕 | 進入批次詳情 | — | 所有狀態 |
| E-RUN-03 | 執行計算 | 觸發 Saga 薪資計算 | `POST /api/v1/payroll-runs/{id}/execute` | DRAFT |
| E-RUN-04 | 送審按鈕 | 提交審核 | `PUT /api/v1/payroll-runs/{id}/submit` | COMPLETED |
| E-RUN-05 | 刪除按鈕 | 刪除草稿批次 | `DELETE /api/v1/payroll-runs/{id}` | DRAFT |

### 4.2 Tab 頁籤

| Tab | 內容 | 元件 |
|:---|:---|:---|
| 員工明細 | 每位員工的薪資明細列表 | `Table` |
| 計算日誌 | Saga 流程各步驟執行記錄 | `Timeline` |

### 4.3 操作事件規格

| 事件 ID | 觸發元素 | 行為 | 對應 API |
|:---|:---|:---|:---|
| E-BATCH-01 | 員工詳情按鈕 | 展開薪資明細 | `GET /api/v1/payroll-runs/{id}/payslips/{empId}` |
| E-BATCH-02 | 執行計算 | 觸發 Saga (202 Accepted) | `POST /api/v1/payroll-runs/{id}/execute` |
| E-BATCH-03 | 送審按鈕 | 提交審核 | `PUT /api/v1/payroll-runs/{id}/submit` |

## 6. HR04-P04 ~ P08 簡要規格

### HR04-P04 薪資結構設定

| 項目 | 說明 |
|:---|:---|
| 頁面類型 | 列表頁 + 編輯 Modal (HR04-M01) |
| 主要功能 | 薪資制度查詢、結構編輯（項目配置、稅保設定） |
| 關鍵 API | `GET/POST/PUT /api/v1/salary-structures` |

### HR04-P05 薪資項目設定

| 項目 | 說明 |
|:---|:---|
| 頁面類型 | 列表頁 + 編輯 Modal (HR04-M02) |
| 主要功能 | 薪資項目維護：底薪、津貼、獎金、扣項 |
| 欄位 | 項目代碼、名稱、類型(加項/扣項)、計算規則、比例 |
| 關鍵 API | `GET/POST/PUT/DELETE /api/v1/payroll-items` |

### HR04-P06 計薪審核

| 項目 | 說明 |
|:---|:---|
| 頁面類型 | 列表頁 |
| 主要功能 | 審核待核准批次，核准或駁回 |
| 操作 | 核准 → `PUT /api/v1/payroll-runs/{id}/approve`；駁回 → 輸入原因 |
| 關鍵 API | `GET /api/v1/payroll-runs?status=SUBMITTED` |

### HR04-P07 銀行轉帳檔管理

| 項目 | 說明 |
|:---|:---|
| 頁面類型 | 功能頁 |
| 主要功能 | 產生銀行薪轉媒體檔、驗證帳號、下載檔案 |
| 操作流程 | 選擇已核准批次 → 驗證帳號 → 產生檔案 → 下載 |
| 關鍵 API | `POST /api/v1/payroll-runs/{id}/bank-transfer` |

### HR04-P08 計薪歷史查詢

| 項目 | 說明 |
|:---|:---|
| 頁面類型 | 列表頁 |
| 主要功能 | 依員工查詢歷史薪資單 |
| 搜尋條件 | 員工姓名/工號、月份範圍 |
| 關鍵 API | `GET /api/v1/payslips?employeeId={id}` |
