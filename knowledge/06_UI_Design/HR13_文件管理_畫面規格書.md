### 3.2 搜尋/篩選條件

| 條件 | 欄位類型 | 選項來源 | API 參數 |
|:---|:---|:---|:---|
| 檔案名稱 | `Input` | — | `fileName` (LIKE) |
| 類型 | `Select` | 合約/規章/證明/薪資單/其他 | `documentType` |
| 擁有者 | `Input` | — | `ownerId` |

### 3.3 操作事件規格

| 事件 ID | 觸發元素 | 行為 | 對應 API |
|:---|:---|:---|:---|
| E-DOC-01 | 上傳文件按鈕 | 開啟上傳 Modal | `POST /api/v1/documents/upload` |
| E-DOC-02 | 下載按鈕 | 下載文件 | `GET /api/v1/documents/{id}/download` |
| E-DOC-03 | 歷程按鈕 | 查看版本歷程 | `GET /api/v1/documents/{id}/versions` |
| E-DOC-04 | 刪除按鈕 | 軟刪除文件 | `DELETE /api/v1/documents/{id}` |

## 4. HR13-P04 文件申請 (ESS)

### 4.1 Wireframe

```
┌──────────────────────────────────────────────────────┐
│ 首頁 / 文件管理 / 文件申請               ← Breadcrumb │
│ 文件申請                                              │
├──────────────────────────────────────────────────────┤
│                                                      │
│  申請類型 *                                          │
│  ┌──────────────────────────────┐                    │
│  │ ☐ 在職證明                   │                    │
│  │ ☐ 離職證明                   │                    │
│  │ ☐ 薪資證明                   │                    │
│  │ ☐ 服務年資證明               │                    │
│  └──────────────────────────────┘                    │
│                                                      │
│  用途說明                                            │
│  ┌──────────────────────────────┐                    │
│  │ 例：銀行貸款用               │                    │
│  └──────────────────────────────┘                    │
│                                                      │
│  份數                                                │
│  ┌──────────────────────────────┐                    │
│  │ 1                            │                    │
│  └──────────────────────────────┘                    │
│                                                      │
│                    [取消]  [提交申請]                  │
│                                                      │
│  流程：提交 → 系統載入範本 → 替換變數 → 產生 PDF     │
│        → 通知員工下載                                │
│                                                      │
└──────────────────────────────────────────────────────┘
```

### 4.2 操作事件規格

| 事件 ID | 觸發元素 | 行為 | 對應 API | UI 反應 |
|:---|:---|:---|:---|:---|
| E-REQ-01 | 提交申請 | 申請文件 | `POST /api/v1/documents/request` | 系統產生 → 發布 DocumentGeneratedEvent |
| E-REQ-02 | 下載按鈕 | 下載已產生文件 | `GET /api/v1/documents/{id}/download` | 下載 PDF |

**薪資單加密：** PAYSLIP 類型文件自動 AES-256 加密，密碼為身分證末 4 碼。

## 6. API 端點對照表

| 方法 | 端點 | 用途 | 對應畫面 |
|:---|:---|:---|:---|
| `GET` | `/api/v1/documents` | 文件列表 | P01 |
| `POST` | `/api/v1/documents/upload` | 上傳文件 | P01 |
| `GET` | `/api/v1/documents/{id}/download` | 下載文件 | P01, P03 |
| `GET` | `/api/v1/documents/{id}/versions` | 版本歷程 | P01 |
| `DELETE` | `/api/v1/documents/{id}` | 刪除文件 | P01 |
| `GET` | `/api/v1/documents/templates` | 範本列表 | P02 |
| `POST` | `/api/v1/documents/templates` | 新增範本 | P02 |
| `PUT` | `/api/v1/documents/templates/{id}` | 更新範本 | P02 |
| `GET` | `/api/v1/documents/my` | 我的文件 | P03 |
| `POST` | `/api/v1/documents/request` | 申請文件 | P04 |
