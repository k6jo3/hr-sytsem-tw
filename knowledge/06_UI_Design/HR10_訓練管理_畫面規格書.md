## 3. HR10-P01 教育訓練（課程管理）

### 3.1 Wireframe

```
┌──────────────────────────────────────────────────────┐
│ 首頁 / 教育訓練                          ← Breadcrumb │
│ 教育訓練                              [+ 新增課程]    │
├──────────────────────────────────────────────────────┤
│ ┌──────────┐ ┌──────────┐ ┌──────────┐              │
│ │ 課程名稱 │ │ 類別 [▼] │ │ 狀態 [▼] │              │
│ │ [______] │ │ [全部  ] │ │ [全部  ] │              │
│ └──────────┘ └──────────┘ └──────────┘              │
│                              [搜尋] [重置]           │
├──────────────────────────────────────────────────────┤
│ ┌────┬────────┬──────┬──────┬──────┬──────┬────┐    │
│ │ #  │ 課程名 │ 類別 │ 時數 │ 報名 │ 狀態 │操作│    │
│ ├────┼────────┼──────┼──────┼──────┼──────┼────┤    │
│ │ 1  │資安訓練│法定  │  8h  │15/30 │報名中│編輯│    │
│ │ 2  │新人訓練│必修  │ 16h  │20/20 │已額滿│查看│    │
│ │ 3  │AI應用  │選修  │  4h  │ 8/50 │草稿  │發佈│    │
│ └────┴────────┴──────┴──────┴──────┴──────┴────┘    │
├──────────────────────────────────────────────────────┤
│                              第 1-20 筆，共 30 筆    │
└──────────────────────────────────────────────────────┘
```

### 3.2 操作事件規格

| 事件 ID | 觸發元素 | 行為 | 對應 API |
|:---|:---|:---|:---|
| E-CRS-01 | 新增課程按鈕 | 開啟課程表單 Modal | `POST /api/v1/training/courses` |
| E-CRS-02 | 編輯按鈕 | 編輯課程 | `PUT /api/v1/training/courses/{id}` |
| E-CRS-03 | 發佈按鈕 | 開放報名 | `PUT /api/v1/training/courses/{id}/publish` |
| E-ENR-01 | 報名按鈕 | 員工報名課程 | `POST /api/v1/training/enrollments` |
| E-ENR-02 | 審核通過 | 通過報名審核 | `PUT /api/v1/training/enrollments/{id}/approve` |

### 3.3 課程狀態流

```
DRAFT → OPEN → FULL → IN_PROGRESS → COMPLETED
草稿    報名中   已額滿   進行中        已完成
```
