## 1. 設計哲學

### 1.1 四大原則

| 原則 | 說明 | 實踐範例 |
|:---|:---|:---|
| **一致性** | 元件樣式、操作方式、回饋機制全系統統一 | 所有列表頁共用 `SearchForm` + `Table` + `Pagination` 結構 |
| **效率優先** | 減少操作步驟，快速完成任務 | 表格行內操作、快速搜尋、鍵盤快捷鍵 |
| **漸進揭露** | 只顯示當前必要資訊，進階功能按需展開 | 搜尋表單基本/進階欄位展開機制 |
| **錯誤容忍** | 操作可撤銷、刪除需確認、表單可暫存 | Modal.confirm 二次確認、表單驗證即時回饋 |

### 1.2 設計語言

- **元件庫**：Ant Design 5.x — 企業級 UI 元件
- **圖表庫**：Apache ECharts — 報表視覺化
- **圖標系統**：@ant-design/icons — 統一圖標風格
- **中文化**：antd/locale/zh_TW — 全系統繁體中文

## 2. Design Token

### 2.1 色彩系統

```
ConfigProvider theme.token（實際程式碼：App.tsx L64-69）
```

#### 主色調

| Token | 值 | 用途 |
|:---|:---|:---|
| `colorPrimary` | `#667eea` | 主色，品牌色，按鈕/連結/選中態 |
| Primary Hover | `#7c91ee` | 主色懸停（自動計算） |
| Primary Active | `#5269d4` | 主色點擊（自動計算） |

#### 語意色

| 語意 | 色值 | 用途 |
|:---|:---|:---|
| Success | `#52c41a` (Ant Design 預設) | 成功訊息、啟用狀態、已核准 |
| Warning | `#faad14` | 警告、留職停薪、處理中 |
| Error | `#ff4d4f` | 錯誤、終止、已駁回 |
| Info | `#1677ff` | 資訊提示、待審核 |

#### 中性色

| 用途 | 色值 |
|:---|:---|
| 標題文字 | `rgba(0, 0, 0, 0.88)` |
| 內文文字 | `rgba(0, 0, 0, 0.65)` |
| 輔助文字 | `rgba(0, 0, 0, 0.45)` |
| 邊框 | `#d9d9d9` |
| 分隔線 | `#f0f0f0` |
| 背景（頁面） | `#f0f2f5` |
| 背景（元件） | `#ffffff` |

### 2.2 字型

```
fontFamily: "'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif"
```

| 用途 | 大小 | 行高 | 粗細 |
|:---|:---|:---|:---|
| 頁面標題 | 20px | 28px | Bold (700) |
| 區塊標題 | 16px | 24px | Semi-Bold (600) |
| 正文 | 14px | 22px | Regular (400) |
| 輔助文字 | 12px | 20px | Regular (400) |
| 表格內容 | 14px | 22px | Regular (400) |

### 2.3 間距系統

以 **4px** 為基礎單位，採用 8 倍數遞增：

| Token | 值 | 使用場景 |
|:---|:---|:---|
| `spacing-xs` | 4px | 圖標與文字間距 |
| `spacing-sm` | 8px | 表單項間距 |
| `spacing-md` | 16px | 區塊內部間距 |
| `spacing-lg` | 24px | Content padding、區塊間距 |
| `spacing-xl` | 32px | 頁面大區塊間距 |

### 2.4 圓角

| Token | 值 | 使用場景 |
|:---|:---|:---|
| `borderRadius` | 8px | 按鈕、卡片、輸入框、Modal |
| `borderRadiusSM` | 4px | Tag、小型元件 |
| `borderRadiusLG` | 12px | 大型卡片、登入框 |

### 2.5 陰影

| 等級 | 值 | 使用場景 |
|:---|:---|:---|
| 陰影-1 | `0 2px 8px rgba(0,0,0,0.1)` | Header、卡片 |
| 陰影-2 | `0 4px 12px rgba(0,0,0,0.15)` | Dropdown、Popover |
| 陰影-3 | `0 8px 24px rgba(0,0,0,0.2)` | Modal、Drawer |

### 2.6 動畫

| 類型 | 時長 | 緩動函數 |
|:---|:---|:---|
| 快速回饋 | 0.1s | ease-out |
| 狀態切換 | 0.2s | ease-in-out |
| 展開/收起 | 0.3s | cubic-bezier(0.645, 0.045, 0.355, 1) |
| 頁面轉場 | 0.3s | ease |

## 5. 共用元件規格庫

### 5.1 PageLayout

| 屬性 | 說明 |
|:---|:---|
| **用途** | 包裹所有受保護頁面，提供統一的 Header + Sider + Content 結構 |
| **檔案** | `frontend/src/shared/components/PageLayout.tsx` |
| **Props** | `children: ReactNode` |
| **整合** | ProtectedRoute 自動包裹 PageLayout（`requireLayout` 預設 true） |

### 5.2 SearchForm

| 屬性 | 說明 |
|:---|:---|
| **用途** | 列表頁統一搜尋表單，支援基本/進階欄位展開 |
| **檔案** | `frontend/src/shared/components/SearchForm.tsx` |

**Props：**

| Prop | 類型 | 預設值 | 說明 |
|:---|:---|:---|:---|
| `items` | `SearchFormItem[]` | 必填 | 表單項目定義 |
| `onSearch` | `(values) => void` | 必填 | 搜尋回調 |
| `onReset` | `() => void` | — | 重置回調 |
| `showExpand` | `boolean` | `true` | 是否顯示展開按鈕 |
| `defaultExpanded` | `boolean` | `false` | 預設是否展開進階欄位 |
| `loading` | `boolean` | — | 搜尋中狀態 |
| `initialValues` | `Record<string, unknown>` | — | 初始值 |

**SearchFormItem：**

| 欄位 | 類型 | 說明 |
|:---|:---|:---|
| `name` | `string` | 欄位名稱 |
| `label` | `string` | 標籤 |
| `type` | `'input' \| 'select' \| 'datePicker' \| 'rangePicker' \| 'custom'` | 欄位類型 |
| `placeholder` | `string` | 佔位文字 |
| `advanced` | `boolean` | 是否為進階欄位（預設摺疊） |
| `render` | `() => ReactNode` | 自訂渲染函數 |
| `colSpan` | `number` | 欄寬（預設 8，即 1/3 寬） |

**佈局規則：**
- Row/Col 24 欄柵格系統
- 每個欄位預設佔 8 欄（1/3 寬度），每行最多 3 個欄位
- 操作按鈕區（搜尋/重置/展開收起）獨佔一行

### 5.3 Pagination

| 屬性 | 說明 |
|:---|:---|
| **用途** | 統一分頁元件，與後端 `PageResponse` 整合 |
| **檔案** | `frontend/src/shared/components/Pagination.tsx` |

**Props：**

| Prop | 類型 | 預設值 | 說明 |
|:---|:---|:---|:---|
| `pageInfo` | `{ page, size, totalElements, totalPages }` | 必填 | 分頁資訊 |
| `onChange` | `(PageRequest) => void` | 必填 | 頁碼/筆數變更 |
| `showSizeChanger` | `boolean` | `true` | 顯示筆數選擇器 |
| `pageSizeOptions` | `number[]` | `[10, 20, 50, 100]` | 筆數選項 |
| `showQuickJumper` | `boolean` | `true` | 快速跳頁 |
| `showTotal` | `boolean` | `true` | 顯示總筆數 |
| `align` | `'left' \| 'center' \| 'right'` | `'right'` | 對齊方式 |

**顯示格式：** `第 1-20 筆，共 100 筆`

### 5.4 StatusTag

| 屬性 | 說明 |
|:---|:---|
| **用途** | 根據狀態類型顯示對應色彩標籤 |
| **檔案** | `frontend/src/shared/components/StatusTag.tsx` |

**狀態對應：**

| 類型 (`type`) | 狀態值 | 顏色 | 顯示文字 |
|:---|:---|:---|:---|
| `active` | `ACTIVE` | `green` | 啟用 |
| `active` | `INACTIVE` | `default` | 停用 |
| `approval` | `PENDING` | `processing` | 待審核 |
| `approval` | `APPROVED` | `success` | 已核准 |
| `approval` | `REJECTED` | `error` | 已駁回 |
| `approval` | `CANCELLED` | `default` | 已取消 |
| `employee` | `ACTIVE` | `green` | 在職 |
| `employee` | `RESIGNED` | `default` | 離職 |
| `employee` | `ON_LEAVE` | `orange` | 留職停薪 |
| `employee` | `TERMINATED` | `red` | 終止 |

### 5.5 Loading

| Prop | 類型 | 預設值 | 說明 |
|:---|:---|:---|:---|
| `tip` | `string` | `'載入中...'` | 提示文字 |
| `size` | `'small' \| 'default' \| 'large'` | `'large'` | 尺寸 |

### 5.6 ErrorMessage

| Prop | 類型 | 預設值 | 說明 |
|:---|:---|:---|:---|
| `message` | `string` | 必填 | 錯誤標題 |
| `description` | `string` | — | 錯誤詳情 |
| `onClose` | `() => void` | — | 關閉回調 |

### 5.7 ProtectedRoute

| Prop | 類型 | 預設值 | 說明 |
|:---|:---|:---|:---|
| `children` | `ReactNode` | 必填 | 受保護的頁面內容 |
| `requireLayout` | `boolean` | `true` | 是否包裹 PageLayout |

**行為：** 檢查 `state.auth.isAuthenticated`，未登入時重導至 `/login` 並保存 `location.state`。

## 7. 表格規範

### 7.1 通用規則

| 規則 | 說明 |
|:---|:---|
| 序號欄 | 寬度 60px，置中 |
| 操作欄 | 寬度 120~180px，固定右側 (`fixed: 'right'`) |
| 文字欄 | 超長文字以 `ellipsis: true` 截斷 |
| 狀態欄 | 使用 `StatusTag` 元件 |
| 日期欄 | 格式 `YYYY-MM-DD`，寬度 120px |
| 金額欄 | 右對齊，千分位分隔，寬度 120px |

### 7.2 操作欄規則

| 操作 | 顯示方式 | 確認機制 |
|:---|:---|:---|
| 查看/詳情 | `Button type="link"` | 無 |
| 編輯 | `Button type="link"` | 無 |
| 刪除 | `Button type="link" danger` | `Modal.confirm` 二次確認 |
| 更多操作 | `Dropdown` 下拉選單 | 依操作性質決定 |

### 7.3 空資料狀態

- 使用 Ant Design `Table` 內建的 `locale.emptyText`
- 顯示：空狀態圖示 + `暫無資料`

### 7.4 批次操作

- 表格支援 `rowSelection` 多選
- 批次操作列顯示在表格上方
- 顯示已選筆數與可用操作按鈕

### 9.1 按鈕規範

**按鈕排列順序：** 取消（左）→ 確認（右）

### 9.2 訊息回饋

| 類型 | Ant Design API | 使用場景 | 持續時間 |
|:---|:---|:---|:---|
| 成功提示 | `message.success()` | 操作成功 | 3 秒自動關閉 |
| 錯誤提示 | `message.error()` | 操作失敗 | 5 秒自動關閉 |
| 警告提示 | `message.warning()` | 注意事項 | 3 秒自動關閉 |
| 通知 | `notification.open()` | 系統通知、重要事件 | 需手動關閉 |
| 確認對話框 | `Modal.confirm()` | 刪除、批次操作等不可逆動作 | 手動關閉 |

### 11.4 共用型別參考

```typescript
// API 回應標準格式
interface StandardApiResponse<T> {
  success: boolean;
  code: string;
  message: string;
  data: T | null;
  timestamp: string;
  traceId?: string;
}

// 分頁回應
interface PageResponse<T> {
  items: T[];
  page: number;
  size: number;
  totalElements: number;
  totalPages: number;
  hasNext: boolean;
  hasPrevious: boolean;
}

// 分頁請求
interface PageRequest {
  page?: number;
  size?: number;
  sortBy?: string;
  sortDirection?: 'ASC' | 'DESC';
}

// 非同步狀態
type AsyncStatus = 'idle' | 'loading' | 'success' | 'error';
interface AsyncState<T> {
  status: AsyncStatus;
  data: T | null;
  error: string | null;
}

// 業務狀態
type ActiveStatus = 'ACTIVE' | 'INACTIVE';
type ApprovalStatus = 'PENDING' | 'APPROVED' | 'REJECTED' | 'CANCELLED';
type EmployeeStatus = 'ACTIVE' | 'RESIGNED' | 'ON_LEAVE' | 'TERMINATED';
```
