## 3. HR05-P01 勞健保加退保

### 3.1 頁面概述

| 項目 | 說明 |
|:---|:---|
| **用途** | 查詢員工投保記錄、手動加保/退保、產生申報檔案 |
| **使用角色** | HR |
| **進入方式** | 側邊選單 → 保險管理 → 勞健保加退保 |
| **頁面類型** | 列表頁 |

### 3.3 搜尋/篩選條件

| 條件 | 欄位類型 | 選項來源 | API 參數 |
|:---|:---|:---|:---|
| 姓名/工號 | `Input` | — | `keyword` |
| 保險類別 | `Select` | 勞保/健保/勞退 | `insuranceType` |
| 狀態 | `Select` | 投保中/已退保 | `status` |

### 3.4 操作事件規格

| 事件 ID | 觸發元素 | 行為 | 對應 API |
|:---|:---|:---|:---|
| E-INS-01 | 手動加保按鈕 | 開啟 HR05-M01 | — |
| E-INS-02 | 調整按鈕 | 開啟級距調整 Modal (HR05-M02) | — |
| E-INS-03 | 產生申報按鈕 | 產生勞保局/健保局申報檔 | `POST /api/v1/insurance/reports/generate` |
| E-INS-04 | Modal 確認加保 | 建立投保記錄 | `POST /api/v1/insurance/enrollments` |
| E-INS-05 | Modal 確認調整 | 調整投保級距 | `PUT /api/v1/insurance/enrollments/{id}/adjust` |
