## ⚠️ 驗證階段：執行合約測試（受阻）

### 目標

執行 29 個合約測試，驗證實作是否符合合約規格。

### 遇到問題

**記憶體不足：** Maven 測試編譯階段失敗

```
OpenJDK 64-Bit Server VM warning:
INFO: os::commit_memory(0x0000000710200000, 58720256, 0) failed;
error='分頁檔太小，無法完成此操作。' (DOS error/errno=1455)

# There is insufficient memory for the Java Runtime Environment to continue.
# Native memory allocation (mmap) failed to map 58720256 bytes.
```

### 解決方案

**推薦方案：增加 Windows 虛擬記憶體**

1. 開啟「系統內容」→「進階系統設定」
2. 「效能」→「設定」→「進階」→「虛擬記憶體」→「變更」
3. 設定分頁檔大小：
   - 初始大小：4096 MB (4GB)
   - 最大值：8192 MB (8GB)
4. 重新開機

**替代方案：**
- 使用 IDE（IntelliJ IDEA/Eclipse）執行測試
- 分批執行測試（逐一執行測試類別）
- 建立 `.mvn/jvm.config` 限制記憶體使用
