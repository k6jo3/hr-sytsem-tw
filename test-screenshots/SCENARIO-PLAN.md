# HRMS 50+ 情境測試計畫

## 情境設計原則
- 每個情境包含多步驟操作
- 涵蓋正常流程 + 異常操作 + 邊界情況
- 從不同角色視角測試（老闆看獲利、主管看成本、員工看薪水）
- 流程可以亂序（使用者不一定照流程操作）

---

## 模組一：HR01 帳號與權限 (5 情境)

### S01 - 管理員建立新使用者帳號
角色：admin → 建立帳號 → 指派角色 → 驗證新帳號可登入

### S02 - 錯誤登入與帳號鎖定
角色：任意 → 連續輸入錯密碼 → 驗證帳號鎖定 → 解鎖

### S03 - 角色權限驗證
角色：employee → 嘗試訪問 admin 頁面 → 驗證被拒絕

### S04 - 修改密碼流程
角色：employee → 個人資料 → 修改密碼 → 重新登入

### S05 - 使用者資料查詢與篩選
角色：admin → 使用者列表 → 搜尋 → 篩選角色 → 篩選狀態

---

## 模組二：HR02 組織與員工 (5 情境)

### S06 - 建立部門與編制
角色：hr_admin → 建立部門 → 設定主管 → 設定編制人數

### S07 - 新增員工完整流程

### S08 - 員工異動（調部門）

### S09 - 員工離職流程
角色：hr_admin → 員工列表 → 設定離職 → 驗證狀態變更

### S10 - 組織架構查看（老闆視角）
角色：admin → 部門與編制 → 查看組織樹 → 確認人力配置

## 模組三：HR03 考勤管理 (7 情境)

### S11 - 員工完整打卡流程（含遲到早退）

### S12 - 員工請假申請流程
角色：employee → 申請請假 → 選擇假別 → 填寫日期 → 送出

### S13 - 主管審核請假
角色：manager → 差勤審核中心 → 請假審核 Tab → 核准/駁回

### S14 - 員工加班申請流程
角色：employee → 請假加班申請 → 選擇加班 → 填寫時數 → 送出

### S15 - 補卡申請流程

### S16 - 班別管理（HR 視角）
角色：hr_admin → 班別管理 → 新增班別 → 設定上下班時間

### S17 - 月結作業（HR 視角）
角色：hr_admin → 月結作業 → 選擇月份 → 執行月結 → 確認結果

## 跨模組整合情境 (4 情境)
