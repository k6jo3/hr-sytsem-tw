### S01 - 管理員建立新使用者帳號

- **步驟：**
  1. 登入 admin / Admin@123
  2. 側邊欄 → 帳號與權限 → 使用者管理
  3. 點擊「新增使用者」按鈕
  4. 填寫：帳號、顯示名稱、Email、選擇角色
  5. 點擊「確定」送出
